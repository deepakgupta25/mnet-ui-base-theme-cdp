# mnet-ui-base-theme-cdp

Media.net Theme CDP

## Installation

```
  $ yarn add mnet-ui/mnet-ui-base-theme-cdp
```

## Usage

```
import { MnetUIBase } from 'mnet-ui-base';
import { cdp } from 'mnet-ui-base-theme-cdp';

<MnetUIBase theme={cdp} />
```
